Ecosystem

Soil moisture sensor

Measures substrate moisture so irrigation triggers only when needed.

  • Placement: mid-depth in the pot.
  • Output: analog 0–1023 (simple threshold).
  • Care: clean probes monthly to reduce corrosion/salts.

ESP8266

Wi-Fi microcontroller that runs control logic and the local web server.

  • Reads sensor → decides → actuates valve.
  • Stores per-species thresholds.
  • Optional deep-sleep to save power.

Relay + valve

Opens water flow in short pulses to avoid overwatering.

  • Pulse: 1–5 s, with cooldown.
  • Use flyback diode when driving coils.
  • Size valve to your tubing flow rate.

LCD 1602

Local status at a glance.

  • Moisture %, active profile.
  • Last irrigation timestamp.
  • I²C backpack for simple wiring.

Web UI

Runs on the ESP8266 over your local network.

  • Select species and thresholds.
  • Manual override and status.
  • Basic alerts on page.

Plant profiles

Predefined moisture ranges by species to reduce trial and error.

  • Basil: higher moisture; short pulses.
  • Lettuce: medium moisture; frequent checks.
  • Tomato: lower moisture; longer cooldowns.

Prototype assembly

Build steps

  1. Install the moisture sensor halfway into the substrate.
  2. Connect the valve/relay to the irrigation line and to the power source.
  3. Mount the ESP8266 and wire sensor + relay per the diagram.
  4. Power the system and check initial status on the LCD.
  5. From the web UI, select the plant profile and test pulse irrigation.
